# bindings-parity-allow.txt — intentional exemptions for check-bindings-parity.
#
# The parity gate (scripts/check-bindings-parity.sh) requires every language
# binding to wrap every symbol declared in the native-trace tier headers (the
# TIER_HEADERS default: asmtest_hwtrace.h, asmtest_drtrace.h, asmtest_trace_auto.h,
# asmtest_ptrace.h, asmtest_codeimage.h) — the surface that is uniformly mirrored
# across all ten bindings and the one most often extended "across all bindings"
# in a single change. List here the (binding, symbol) pairs that are deliberately
# NOT wrapped.
#
# Syntax:  <lang|ALL> <symbol>          # everything after '#' is a comment
#   ALL   exempts the symbol for every binding.
#   <lang>  exempts it for one binding, where <lang> is one of:
#           python cpp rust zig node java dotnet ruby lua go
# Blank lines and lines beginning with '#' are ignored.
#
# Stale entries fail the gate (the binding actually wraps the symbol now, or the
# symbol left the tier headers), so this file cannot rot silently — remove an
# entry once it is no longer needed.
